# Swag Lab Test Case

## Project Overview

This project is designed to automate the online shopping experience using Cypress. It simulates a user logging into an online shop, purchasing items, and logging out, all while validating the necessary elements throughout the process.

## Features

- User login functionality
- Product selection and addition to cart
- Checkout process
- Order confirmation
- Logout functionality

## Test Scenario

### User Story
As a user, I want to purchase items through online shopping.

### Test Case
The user wants to log in to an online shop in incognito mode.
